## 解析

代码:

```rust
include!("/flag")

fn main() {
    println!("hello");
}
```

`include!`宏会包含本地文件，直接嵌入上下文一起编译

就是一个简单的本地任意文件包含，直接包含文件`/flag`即可，根据报错能看到flag具体的内容
